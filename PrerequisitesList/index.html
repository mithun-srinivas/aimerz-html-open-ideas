Prerequisites List

A simple tool to explore the basics required for learning various technologies taught in Full Stack Web Development with AI course by Aimerz.

HTML
  • No Prior Knowledge Required: HTML is beginner-friendly and doesnâ€™t require prior programming knowledge.
CSS
  • HTML: Basic understanding of HTML is required since CSS is used to style HTML elements.
Tailwind CSS
  • HTML and CSS: Proficiency in HTML and CSS basics since Tailwind is a utility-first CSS framework.
  • Understanding of Responsive Design: Familiarity with responsive design principles, as Tailwind makes it easy to create responsive layouts.
JavaScript
  • HTML and CSS: Basic knowledge of HTML and CSS to manipulate web pages.
React
  • JavaScript Proficiency: Strong understanding of JavaScript, including ES6 features like arrow functions, destructuring, and modules.
  • HTML and CSS: Basic knowledge of HTML and CSS, as React components are written using these technologies.
  • Component-Based Architecture: Familiarity with the concept of components and how they work together in an application.
  • JSX Syntax: Familiarity with JSX (JavaScript XML), as it's used to write React components.
Node.js
  • JavaScript: Proficiency with JavaScript, as Node.js is JavaScript-based.
  • Asynchronous Programming: Understanding callbacks, promises, and async/await, as Node.js uses asynchronous I/O.
  • Command Line Basics: Familiarity with using the command line, since Node.js is often run via terminal commands.
  • Basic HTTP/Networking: Understanding how servers work, and familiarity with the HTTP protocol.
Express.js
  • JavaScript: Solid understanding of JavaScript fundamentals, including ES6 features.
  • Node.js: Experience with Node.js, since Express is a framework built on top of it.
  • HTTP and REST APIs: Knowledge of how HTTP works, along with RESTful API concepts.
  • Basic Routing: Understanding how to navigate URLs and route between them in web applications.
MongoDB
  • Basic understanding of databases: Understand what databases are and why they're used.
  • JSON: Knowledge of JSON format, since MongoDB stores data as BSON, which is a binary representation of JSON.
  • JavaScript: Understanding JavaScript can be helpful, especially when using MongoDB with Node.js.
  • CRUD Operations: Familiarity with Create, Read, Update, Delete operations.
WebSockets
  • JavaScript: Knowledge of JavaScript for writing client and server-side code.
  • HTTP vs WebSocket: Understanding the difference between HTTP (request-response model) and WebSocket (full-duplex communication).
  • Networking Basics: Familiarity with TCP/IP and how network connections are established and maintained.
  • Client-Server Architecture: Basic understanding of how communication occurs between client and server.
Langchain
  • Basic Machine Learning Concepts: Understanding machine learning fundamentals and working with models.
  • OpenAI and LLMs: Some familiarity with Large Language Models (LLMs) like GPT-3/4 and APIs like OpenAI.
  • Prompt Engineering: Understanding how to create effective prompts to get desired outputs from language models.
Hugging Face
  • Machine Learning Concepts: Understanding of deep learning and NLP (Natural Language Processing).
  • Transformers: Familiarity with transformer models, as Hugging Face is well known for them.
  • PyTorch or TensorFlow: Basic understanding of these frameworks, as they are often used in Hugging Face models.
  • API Usage: Basic experience in using APIs, especially the Hugging Face API for integrating pretrained models.
